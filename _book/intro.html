A Minimal Book Example

Chapter 2 Introdução

R é uma linguagem de programação muito popular entre pesquisadores e analistas de dados. A linguagem, desenvolvida por volta dos anos 70’s por Ross Ihaka e Robert Gentleman no Departamento de Estatística da Universidade de Auckland, Nova Zelândia, concentra-se na implementação de ferramentas para análise estatística de dados. Atualmente é mantido por uma comunidade de colaboradores voluntários que contribuem com o melhoramento do código fonte da linguagem e com a expansão das suas funcionalidades.

A linguagem, criada como uma alternativa de código aberto por volta dos anos 90’s, é utilizada por estatísticos, cientistas, economistas, biólogos, engenheiros e, mais recentemente, cientistas de dados, como uma ferramenta sofisticada para a análise exploratória e visualização de dados. Ao contrário do que se imagina sobre a linguagem R, ela não se limitada apenas a sessões iterativas, pois devido ao fato de ser uma linguagem de programação, os scripts podem ser criados e compactados em bibliotecas. As mesmas proporcionam funcionalidades extras para a obtenção de resultados mais consistentes e confiáveis.

A linguagem R tem sido uma referência na análise estatística durante muitos anos. A obtenção e distribuição são gratuítas e está sob a Licença Pública Geral (GPL) do projeto colaborativo de Software livre GNU. Dessa forma, sendo um programa de código aberto, R é, portanto, livre, mas acima de tudo, é o resultado do esforço de milhares de pessoas ao redor do mundo que colaboram no seu desenvolvimento. Isso permite solucionar erros de programação (bugs) mais rapidamente e desenvolver pacotes especializados para análises específicas.

A princípio, o primeiro contato com a linguagem pode, eventualmente, causar um certo abalo para aqueles usuários acostumados a manipular programas de computador com menus e ajuda do mouse. O motivo é muito simples: O R é é uma linguagem interpretada, pois os códigos são executados por um interpretador.

Figure 2.1: Primeiro contato com R. Desenho adaptado das ilustrações de Allison Horst.

No entanto, depois de ter superado naturalmente esse obstáculo, esses usuários perceberão que o uso de uma IDE adequada, junto com a utilização de pequenos scripts, facilitá a execução de uma longa seqüência de operações de forma mais eficiente, do que uma fatigante sessão de ”clics” com o mouse. Os scripts permitem repetir facilmente o mesmo procedimento com diferentes conjuntos de dados, além disso, reduz consideravelmente a possibilidade de cometer erros em uma sequência rotineira de operações.

Frequentemente são disponibilizados novos pacotes de R que facilitam o trabalho do analista de dados. Tais pacotes estão disponíveis por data de publicação no site do CRAN (Comprehensive R Archive Network). No site do projeto, os usuários podem encontrar documentação adicional e toda a informação referente à linguagem de programação.

2.1 Porque ler este livro?

Este livro destina-se a usuários com conhecimento básico em análise exploratória de dados que desejam iniciar a gestão e análise de dados espaciais com R. Portanto, não requer nenhum conhecimento prévio deste programa, mas um conhecimento básico do SIG. O livro pretende permitir ao leitor dar os primeiros passos na gestão de R para a análise espacial sem muitos tropeços. Para continuar com aplicativos mais avançados, há um ótimo número de fontes de informação (MELHORAR).

2.2 Estrutura do livro

You can label chapter and section titles using {#label} after them, e.g., we can reference Chapter 2. If you do not manually label them, there will be automatic labels anyway, e.g., Chapter ??.

Figures and tables with captions will be placed in figure and table environments, respectively.

par(mar = c(4, 4, .1, .1))
plot(pressure, type = 'b', pch = 19)

Figure 2.2: Here is a nice figure!

Reference a figure by its code chunk label with the fig: prefix, e.g., see Figure 2.2. Similarly, you can reference tables generated from knitr::kable(), e.g., see Table 2.1.

knitr::kable(
  head(iris, 20), caption = 'Here is a nice table!',
  booktabs = TRUE
)
